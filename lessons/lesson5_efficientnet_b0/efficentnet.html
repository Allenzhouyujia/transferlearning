Compound Scaling Method

EfficientNet uses a principled method to scale up CNNs by uniformly scaling network depth, width, and resolution with a set of fixed scaling coefficients.

depth: d = α^φ
width: w = β^φ
resolution: r = γ^φ

subject to: α · β² · γ² ≈ 2
α ≥ 1, β ≥ 1, γ ≥ 1
🔧
MBConv Architecture

Mobile Inverted Bottleneck: Uses depthwise separable convolutions with inverted residuals and linear bottlenecks for efficient computation.

Expansion Factor: MBConv6 blocks expand channels by 6× in the middle layer, then compress back to save parameters.

🎯
Squeeze-and-Excitation

Channel Attention: SE blocks adaptively recalibrate channel-wise feature responses by explicitly modelling interdependencies between channels.

Performance Boost: Minimal computational overhead with significant accuracy improvements across all stages.

⚖️
Compound Scaling

Balanced Growth: Unlike conventional practice that arbitrarily scales only one dimension, compound scaling uniformly scales all three dimensions.

Efficiency Gains: Achieves better accuracy and efficiency than scaling individual dimensions independently.

🧠
NAS Foundation

Automated Design: Base architecture (B0) discovered through Neural Architecture Search for optimal accuracy-efficiency trade-off.

Scalable Family: B0 serves as foundation for scaling up to B1-B7 variants using compound scaling method.
